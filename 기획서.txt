db 경우의 수정
    카카오맵을 통한 모임위치표시
        class table 조회하여 모임이름으로 핀이름 장소로 핀위치 설정
    모임핀을 클릭시 모임정보 조회
        class table 과 class_info table inner join하여 모임정보 조회
    게시글 조회기능
    board table 조회하여 code 순서대로 보여주기
    게시글 검색기능
    board table에서 제목 및 내용 작성자 날짜를 통해 게시물 검색기능 구현

    모임정보 입력기능
        class table에 모임이름 모임위치 입력
        class table code프라임키 상속받은 class_info table에 모임 새부정보 입력
    
    게시판 글 등록 기능
        board table에 게시글작성
    댓글 등록 기능
        해당 게시글 code 비식벌 상속받은 comment table에 입력 프라임키는 board코드&코멘트코드 복합키
    
    관리자 기능
        관리자 로그인시 jwt방식으로 로그인 유지 관리자 로그인시 모임등록 및 게시글 관리 버튼 추가

    삭제기능
    게시물 삭제기능 비밀번호 입력시 가능
    댓글 삭제기능 비밀번호 입력시 가능
    관리자일시 바로 가능
    모임정보 삭제기능

    수정기능
    게시판 수정 글만 수정가능 <--날짜는 변하지않음
    댓글 내용만 수정 가능<-- 날짜는 변하지않음
    모임정보 수정기능 이름 및 장소는 class table 수정 그외 정보는 class_info table 수정
